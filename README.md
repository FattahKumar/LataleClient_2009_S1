
# LataleClient（相对旧服的新功能改动）

本文仅记录相对 VS2003 原始旧服的新增/替换型功能变化。  
本文不是已确认问题清单，也不是构建迁移手册。  
统计范围为迁移期全量提交中可在当前仓库落证的改动。

## 新功能改动

- 血条模式切换：旧服为单一血条表现；现版支持 `new/old` 血条模式切换，并补充 Boss 多段血量表现链路。
- 选项项替换（亮度 -> 全局 FX 透明度）：旧服使用亮度（Gamma）滑条；现版替换为全局 FX 透明度控制，并统一作用于特效透明表现。
- 稳态帧率节奏控制：旧服采用阈值 `Sleep` 节流；现版采用 200FPS 稳态限帧节奏器（QPC + 高精定时器 + 短自旋）。
- 切图加载链路改造：旧服 Loading/EnterStage 链路耦合较紧；现版按目标 Group 加载并同步补齐，读条按真实阶段推进。
- 地形分组预取机制：旧服偏向全量路径加载；现版引入运行时 Group 预取与预算 Pump，优先保证当前组可用并后台推进。
- 怪物贴图异步流水线：旧服贴图加载对主线程压力更集中；现版接入 WIC 解码、主线程分帧上传与任务队列，降低首遇多贴图怪时的卡顿峰值。
